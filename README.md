# Visiotrainer: Facharzt-Training für Dermatoskopie

Ein interaktives Trainings-Tool für angehende Fachärzte zur Beurteilung dermatoskopischer Befunde. 
Entwickelt mit **.NET MAUI** für Cross-Plattform-Performance und einer **MySQL**-Anbindung für skalierbare Datenverwaltung.

---

## Highlights & Features
- Interaktive Diagnose: Training der dermatoskopischen Beurteilung anhand echter Merkmalsanalysen.
- Cross-Plattform: Eine Codebase für mobile und Desktop-Anwendung (.NET MAUI).
- Datenbank-Integration: Speicherung von Trainingsfortschritten via MySQL.
- Logik-Fokus: Implementierung medizinischer Entscheidungsbäume in C#.

## Tech Stack
- Frontend: .NET MAUI (XAML / C#)
- Backend: C# / .NET
- Datenbank: MySQL
- Architektur: MVVM (Model-View-ViewModel) – falls du das genutzt hast, unbedingt erwähnen!
